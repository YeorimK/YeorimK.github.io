파이썬의 리스트 데이터 타입을 정리한 두 번째 글입니다.   
리스트 연산에 대해 정리해보도록 하겠습니다.

# 1. 리스트 더하기
```python
>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> a + b
[1, 2, 3, 4, 5, 6]
```
문자열과 마찬가지로 리스트도 연산이 가능합니다. + 기호를 통해서 리스트를 이어붙일 수 있습니다. 

<br>

# 2. 리스트 반복하기
```python
>>> a = [1, 2, 3]
>>> a * 3
[1, 2, 3, 1, 2, 3, 1, 2, 3]
```
\* 기호를 통해서 리스트를 반복할 수 있습니다. 

<br>

# 3. 리스트 길이 구하기
```python
>>> a = [1, 2, [1,2]]
>>> len(a)
3
```
리스트의 길이, 즉 리스트 요소의 개수는 len() 함수를 통해서 구할 수 있습니다.
